Memberships
* User should see link of basic plan on Home Page
* User should see link for pro plan on Home Page
* Basic plan should point to basic devise sign up form
* Pro plan  link should point to a custom sign up form

Stripe Integration:
* Users should have a Stripe Customer Token in the users table
* JavaScript should prevent pro form from submitting when a user fills it out,
    and should send the card information to stripe. Stripe will return with a card Token.
* JavaScript should send User fields and card token to our Rails app for validation. Rails app 
    should validate the form field.
*  Once validated, Rails should send user info email, plan ID and card token to Stripe to sefer.
    Stripe will then create a subscription and send us a customer token.
* Rails should save customer token and User Invofmation to our dabase.
